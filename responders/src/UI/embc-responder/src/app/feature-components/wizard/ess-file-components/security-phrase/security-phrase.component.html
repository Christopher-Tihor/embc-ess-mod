Security Phrase

The security phrase is a vital piece of information to ensure an Emergency Support Services (ESS) Responder can verbally verify your identity with your registration.

Enter a memorable security phrase that is over 6 characters in length and does not include any special characters. You may include numbers and letters. The security phrase is not case sensitive.

For security reasons, your security phrase should not include your email address.

Security Phrase Phrase is required. Phrase must be between 6 and 50 characters long. Phrase cannot contain special characters.

Tip: Security Phrase

If you have been evacuated, please share your security phrase with any household members that you are registering. This will enable an Emergency Support Services (ESS) Responder to assist them when you are not able to be present.
